# StudySprint

StudySprint is a modern productivity web application designed to help students stay focused and organized during study sessions.

## Features
- Sprint-based study timer
- Task management system
- Productivity tracking
- Dark and light mode
- Responsive UI
- Interactive animations

## Tech Stack
- React.js
- HTML5
- CSS3
- JavaScript

## Project Purpose
This project was built after completing the “Web Development Fundamentals” course by IBM SkillsBuild to apply frontend development concepts into a real-world mini product.
